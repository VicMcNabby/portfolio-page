Portfolio

The projects below are a few of my favorites. They are presented in chronological order based on when they were created. While I am proud of all of them, I also think they are a good display of just how far I've come in my programming education. I continue to learn more each day, and look forward to what projects and oppurtunities the future holds. Please enjoy!

'Flizz' is a multimedia search engine designed to display the top results for 'books', 'movies', and 'games' of a particular search parameter. Each individual result will display an image and brief description. Also, an Amazon link will be attached to each result as well as media specific links, such as the platforms available for each game, or a link to the trailer for each movie. The goal of this app is to introduce the user to different ways to consume their favorite topics.

'Jello' is a workflow organization app for teams and/or individuals. We envision this app will be used by developers, or any team based environment to maintain organization. 'Jello' will allow the user to see a visual representation of their tasks organized by priority, enable them to create new projects, and assist in assigning tasks/stories to a particular user.. This was a four member group project completed in a weeks time. We set out to create our own version of a 'Trello' type application.

'RegXpress' is real time contest between one or more players where the goal is to accumulate points by solving complex, scenario based word pattern problems using regex. Socket.IO was used in order to enable the realtime communication between multiple users. For our front-end framework, we used AngularJS to bring our app to fruition. This was a group collaboration between myself and three other classmates. We followed the agile workflow and used feature branching on Github to avoid merge conflicts.

This is a hybrid mobile application for Moms to find other Moms within their specified location. The user will be able to find baby/kid friendly activities nearby, as well as family friendly restaurants. There is also an advice section to give and recieve advice with other Moms. Finally the coupon section, powered by the Groupon API, will help the user save money on various family friendly activitites.

'ChaseFilms' is a movie database application that lets the user search through the entire catalog and see detailed information on each film. The user can then rate movies that they have seen and add it to their 'collection' page. If the user hasn't seen a particular film, but is interested in doing so, they can add it to their 'watchlist' page. After they get around to watching it, they then rate it and it is added to their 'collection' page. This app was created with AngularJS.

'Pawfinder' is a web application with the purpose of helping the user find a rescue dog or cat in whichever zipcode they enter. The app displays the top 100 results over 4 pages that contain 25 pets each. Clicking on the individual pet photo will bring up a custom modal which gives much more detailed info on the respective pet as well as all the important contact details. This app was made with AngularJS and utilizes the petfinder API.
